# Getting Started

이 디자인 시스템을 프로젝트에 적용하는 방법을 안내합니다.

## 설치

```bash
pnpm add @my/design-system
```

## 기본 사용법

### 1. CSS 임포트

컴포넌트 스타일과 디자인 토큰(CSS 변수)을 사용하려면 앱 진입점에서 스타일을 임포트합니다.

```tsx
// main.tsx 또는 App.tsx
import '@my/design-system/styles';
```

### 2. 컴포넌트 사용

```tsx
import { Button, TextField, Modal, Tabs, toast, ToastContainer } from '@my/design-system';

function App() {
  return (
    <>
      <Button variant="solid" colorScheme="primary">
        클릭
      </Button>
      <ToastContainer />
    </>
  );
}
```

### 3. 테마 적용 (선택)

`ThemeProvider`로 다크모드 전환과 커스텀 토큰 오버라이드가 가능합니다.

```tsx
import { ThemeProvider } from '@my/design-system';

function App() {
  return (
    <ThemeProvider defaultMode="light">
      {/* 앱 내용 */}
    </ThemeProvider>
  );
}
```

#### 다크모드 전환

```tsx
import { useTheme } from '@my/design-system';

function ThemeToggle() {
  const { mode, toggleMode } = useTheme();
  return (
    <button onClick={toggleMode}>
      현재: {mode === 'light' ? '라이트' : '다크'}
    </button>
  );
}
```

#### 커스텀 토큰 오버라이드

```tsx
<ThemeProvider
  defaultMode="light"
  themeOverride={{
    '--color-primary-500': '#8b5cf6',  // 보라색 브랜드 컬러
    '--radius-md': '16px',              // 더 둥근 모서리
  }}
>
  {/* 브랜드 커스텀 테마 적용 */}
</ThemeProvider>
```

## 설계 원칙

1. **접근성 내장** — WAI-ARIA 패턴 준수, 키보드 내비게이션, 스크린리더 지원
2. **Zero-config** — 기본값만으로 바로 사용 가능, 필요시 커스터마이징
3. **CSS 변수 기반 테마** — 런타임 오버헤드 없이 테마 전환
4. **타입 안전** — 모든 Props에 TypeScript 타입 제공
